O exercício consiste no desenvolvimento de um formulário com duas entradas de números, onde o formulário só pode ser validado quando o segundo número for maior que o primeiro.
Quando o primeiro número for maior, o formulário deve apresentar uma mensagem para o usuário dizendo que a resposta é inválida.
Se válida, o formulário deve apresentar uma mensagem para o usuário dizendo que a resposta é, sim, válida.
